# Tech Context

## Technologies Used

- **Frontend:**
  - React
  - TypeScript
  - Redux Toolkit
  - React Router
  - Tailwind CSS
  - Axios
  - `clsx` for conditional class names
  - `lucide-react` for icons
  - `react-hot-toast` for notifications

- **Backend:**
  - Node.js
  - Express
  - MongoDB
  - Mongoose
  - JSON Web Tokens (JWT)
  - Passport.js

## Development Setup

- The frontend and backend are in separate directories, `frontend` and `backend`, respectively.
- The frontend is a Create React App application.
- The backend is a standard Node.js application.
- Both the frontend and backend have their own `package.json` files and dependencies.

## Technical Constraints

- The application must be compatible with modern web browsers.
- The API should be RESTful and stateless.
